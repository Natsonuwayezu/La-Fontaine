# 🏫 Ecole La Fontaine — School Management System

## Overview
A complete digital resource hub for **Ecole La Fontaine**, Rubavu, Rwanda.  
This project contains teacher's notes, lesson plans, and examinations following the **REB CBC curriculum** for primary levels P4, P5, and P6.

## 📚 Contents

### English Department
- **P4 English** — Teacher's notes and end-of-year exams (Terms 1, 2, 3)
- **P5 English** — Teacher's notes and end-of-year exams (Terms 1, 2, 3)
- **P6 English** — Teacher's notes and end-of-year exams (Terms 1, 2, 3)

### Curriculum
All materials follow the **REB Competence-Based Curriculum (CBC)** structured around:
- Listening & Speaking
- Reading & Writing
- Grammar & Vocabulary
